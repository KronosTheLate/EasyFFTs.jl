## Introductary example
Input:
```
using EasyFFTs
using UnicodePlots
let
    fs = 1000
    duration = 1
    timestamps = range(0, duration, step=1 / fs)

    f = 5
    A = 2
    s = @. A * sin(f * 2π * timestamps)

    plt1 = scatterplot(timestamps, s)

    s_fft = easyfft(s, fs)
    plt2 = scatterplot(s_fft.freq, s_fft.resp .|> abs)
    display(plt1)
    display(plt2)
end
```
